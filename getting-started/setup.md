# Setup

### Requirements

Before starting development with Bura, make sure you have the following installed:

* Java 21
* IntelliJ IDEA
* Git (optional but recommended)

***

### Download Bura

Download the latest Bura development release from the official release page:

[https://github.com/evodevteam/bura/tags](https://github.com/evodevteam/bura/tags)

Download the project files and extract them to a folder on your computer.

***

### Open Project in IntelliJ IDEA

1. Open IntelliJ IDEA.
2. Click File â†’ Open.
3. Select the Bura project folder.
4. Open the project as a Gradle project if prompted.
5. Wait for indexing and dependency setup to finish.

After importing the project:

1. Open the Gradle panel on the right sidebar.
2. Click **Execute Gradle Task**.
3. Search for `runClient`.
4. Click `runClient` to start the development environment.

***

### Run Development Environment

Running `runClient` will launch the Minecraft development environment with Bura loaded.

***

### Verify Setup

Setup is successful if:

* Minecraft launches.
* Bura loads without errors.
* No critical errors appear in the console.
